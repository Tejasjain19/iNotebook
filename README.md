# iNotebook

Website for taking and managing notes using MERN
App Link :- https://tejas-inotebook.netlify.app/
Backend deployed Link :- https://inotebook-backend.glitch.me/

## Tech stack

- ReactJs
- Express JS
- Node JS
- Mongo DB
